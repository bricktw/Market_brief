# data/tw0050.json

50 constituents of the Yuanta Taiwan Top 50 ETF (0050.TW / 元大台灣50), in
Finnhub ticker format (`{TWSE code}.TW`).

**Status (2026-05-18): the TW earnings filter built on this list is currently
inert.** Finnhub's free tier returns an entirely US-only earnings calendar — no
`.TW` symbols (and no exchange suffixes of any kind) appear in the unfiltered
feed; `?symbol=` filtering and `/stock/profile2` return HTTP 403 (premium).
TSMC does not even appear via its US ADR `TSM` in the earnings calendar.

Per CLAUDE.md ("Finnhub free-tier coverage of TW tickers" risk) the decision is
to **accept this gap and not spike to the paid tier**. `filterEarningsTw0050()`
is wired and correct; it will start producing results automatically if/when a
TW-capable earnings source is added (paid Finnhub, TWSE, or another provider).

**Maintenance:** the 0050 basket is reconstituted quarterly. This list is an
approximate as-of-2026-05 snapshot of the well-established large-cap
constituents, sufficient for v1 (CLAUDE.md open question accepts quarterly
manual refresh). Re-verify against the official Yuanta 0050 holdings before
relying on exact membership.
